== Description ==

**Important:** As TipiT is discontinued, this plugin is no longer maintained.

Makes it easy to display a link to your TipiT tipjar in posts. Also includes a sidebar widget and a customizable shortcode.

It adds the following features:

* Options to add a link to your TipiT tipjar under all posts
* A widget with customizable text
* A shortcode with customizable text

Read more about this plugin on the [TipiT Suite plugin](http://coenjacobs.net/wordpress/plugins/tipit-suite) page.

== Installation ==

1. Upload the `tipit-suite` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the shortcode in a post or page
1. Place the widget inside one (or more) sidebar
1. Set the settings for displaying the link underneath posts in Options panel provided

== Frequently Asked Questions ==

= Are there any required settings in the shortcode? =

Yes, `url` is required but `text` has a default value
